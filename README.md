🚀 Smart Analytics & Anomaly Detection System

Developed a full-stack web application that integrates modern frontend design with backend intelligence to deliver real-time analytics and anomaly detection.

Built a responsive user interface using React for seamless user interaction and data visualization
Designed and implemented a Flask-based REST API to handle authentication, data processing, and business logic
Managed structured data efficiently using PostgreSQL, ensuring scalability and reliability
Integrated a Machine Learning model (Isolation Forest) to detect anomalies in datasets and generate actionable insights
Enabled real-time data flow between frontend, backend, and ML components for dynamic decision-making
Focused on clean architecture, modular design, and performance optimization
